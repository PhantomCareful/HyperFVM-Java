# 功能列表
### 已实现的功能
- **防御卡数据库**：目前游戏内已有的所有卡片均已入库（部分纪念卡除外）。输入卡片名称，即可查询该卡片的基础信息、星级数据、技能数据以及其他大部分有关此卡片的数据信息。其中：**金卡数据目前已支持查看分解兑换数据，生肖卡和星座卡后续将逐步支持。**
- **数据图合集**：目标是打造成最全面的数据图合集。目前已收录So陌路の大佬制作的所有数据图，以及作者自己制作的一些数据图，图片保存在本地，方便随身查看。
- **温馨礼包自动领取**：支持不限量绑定温馨礼包链接，并可设置让App每天自动领取。
- **信息仪表盘**：展示各类重要信息，包括：
  - ***双倍双爆信息***
  - ***月签道具数量提醒***
  - ***温馨礼包领取状态***
  - ***施肥活动信息***
  - ***美食悬赏活动信息***
  - ***月末提醒清空积分和金券***
  - ***更新公告**：快捷跳转B站官方账号的最新更新公告*
  - ***米鼠的图**：保存部分由提拉米鼠机器人生成的图片*
- **美食数据站**：收录各类实用网站入口，方便快捷跳转。包括：
  - ***提拉米鼠使用指南***
  - ***陌路の综合数据表***
  - ***FAA米苏物流***
  - ***卡片鼠军对策表***
  - ***查黑系统**：查询某个QQ号是否有行骗记录*
  - ***强卡最优路径计算器***
  - ***FVM查黑系统（提供本地UI，无需跳转）***
- **实用功能**
  - ***今日运势**：一个快速播放不同星级海星的GIF动图，可以暂停和播放。*
  - ***威望储备计算器**：方便让威望币爆仓的同学知道自己一共有多少威望。*
- **在线功能**：通过云端仓库，实现App和图片资源实时更新。

### 计划实现的功能
只展示大功能，开发不分先后顺序
- 防御卡数据库：查询生肖卡、星座卡时支持展示分解兑换信息。
- 防御卡数据库：查询普通卡时，支持展示相应的配方材料、技能书的掉落信息（如果有的话）。
- 防御卡数据库：内容支持使用Markdown格式展示。目前融合卡和金卡已经支持。
- 防御卡数据库：持展示其对应食神谱的图片。目前融合卡和金卡已经支持。
- 防御卡数据库：查询卡片时，支持展示其获取方式（周氪、消费等等）。
- UI引擎适配Miuix

# 一些小细节
- 使用Material You组件构建UI界面，支持Monet动态主题，更加简洁优雅。
- 所有界面的顶部标题栏、底部导航栏、悬浮按钮均添加模糊材质。
- 完美适配全场景底部小横条沉浸，不要像某些国民级App那样做得稀烂🙄
- 支持预见式返回动画，在部分系统上（原生Android14+、OneUI、ColorOS16等）能够体验到非常跟手的过渡动画。

# 安装
### 支持的系统环境
- ✅Android：支持Android12-16。
- ✅HarmonyOS：支持鸿蒙3-4.3，鸿蒙Next需要在卓易通环境下才能使用，请自行测试。
- ❌不支持Windows、iOS，感谢大家的理解。

### 如何下载/升级
- 在此仓库的Releases处选择需要的.apk安装文件下载即可。
- 腾讯频道：搜索【HyperFVM交流社区】，在【更新公告】版块可以找到最新的网盘链接，选择最新的.apk安装文件下载即可。
- App内：在【关于软件】中，点击【检查更新】按钮，可将App升级到最新版本。

### 版本选择
- Beta版：想要第一时间体验到新功能的同学可以选择Beta版，但需要注意可能会遇到未知Bug。
- Release版：推荐所有同学下载/更新。
