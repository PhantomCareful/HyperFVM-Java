# 功能列表
### 已实现的功能
- **仪表盘**：展示各类重要信息，包括：
  - ***公告栏：内容通过云端仓库下发，用于发布重要通知***
  - ***双爆信息：查看今天为限时双爆还是全天双爆，内容格式借鉴了提拉米鼠机器人***
  - ***温馨礼包自动领取：支持不限量绑定温馨礼包链接，并可设置让App每天自动领取。***
  - ***每日签到：展示签到进度，防止漏签***
  - ***施肥活动：展示施肥活动的起止时间和进度***
  - ***抢红包：如果当天有抢红包活动，会进行提醒***
  - ***美食悬赏：展示悬赏任务的起止时间和进度***
  - ***跨服助人为乐：展示助人为乐活动的起止时间和今天是第几天***
  - ***三岛福利：展示三岛活动的起止时间和今天是第几天***
  - ***美食大赛：展示美食大赛的起止时间和今天是第几天***
  - ***二转打折：展示二转打折活动的起止时间、今天是第几天、参与的卡片名单***
  - ***更新公告：跳转查看美食B站官号发布的最新更新公告，链接通过云端仓库更新下发，若没有第一时间更新还请谅解***
  - ***日氪：展示日氪的起止时间和今天该有多少个道具***
  - ***欢乐假期：展示欢乐假期的起止时间和今天是第几天***
  - ***百万消费：展示百万消费的起止时间和今天是第几天***
- **数据相关**
  - **防御卡全能数据库**：目前游戏内已有的所有卡片均已入库（部分纪念卡除外）。输入卡片名称，即可查询该卡片的基础信息、星级数据、技能数据、分解兑换数据以及其他大部分有关此卡片的数据信息。
  - **增幅卡名单**：快捷查看防御卡数据图中，投掷增幅卡和名单类增幅卡的增幅名单信息。
  - **数据图合集**：收录陌路老师制作的所有数据图，也包含一些我自制的数据图和从其他大佬那里看到的数据。数据图将通过云端仓库的方式不断更新，App本身不包含这些图片资源，因此首次使用时您需要根据软件内的引导下载图片资源，之后便可正常查看了。
  - **米鼠的图**：可以查看部分由提拉米鼠生成的最新图片。图片将通过云端仓库的方式不断更新，App本身不包含这些图片资源，因此首次使用时您需要根据软件内的引导下载图片资源，之后便可正常查看了。具体包括：
    - *周氪图*
    - *日氪图*
    - *消费图(3张)*
    - *积分兑换图*
    - *三岛兑换图*
    - *假期兑换图*
    - *大赛任务图(4张)*
- **美食数据站**：收录各类实用网站入口，方便快捷跳转。包括：
  - ***提拉米鼠使用指南***
  - ***陌路の综合数据表***
  - ***FAA米苏物流***
  - ***卡片鼠军对策表***
  - ***查黑系统**：查询某个QQ号是否有行骗记录*
  - ***巅峰对决部分机制解析***
  - ***强卡最优路径计算器***
  - ***宝石最优路径计算器***
- **实用功能**
  - ***今日运势**：一个快速播放不同星级海星的GIF动图。*
  - ***威望储备计算器**：方便让威望币爆仓的同学知道自己一共有多少威望。*
- **在线功能**：通过云端仓库，实现App和图片资源实时更新。

# 一些小细节
- 使用Material You组件构建UI界面，支持Monet动态主题，更加简洁优雅。
- 顶部标题栏、底部导航栏、悬浮按钮、弹窗等场景均添加模糊材质。
- 完美适配全场景底部小横条沉浸，不要像某些国民级App那样做得稀烂🙄
- 支持预见式返回动画，在部分系统上（原生Android14+、OneUI、ColorOS16等）能够体验到非常跟手的过渡动画。

# 安装
### 支持的系统环境
- ✅Android：支持Android12-16。
- ✅HarmonyOS：支持鸿蒙3-4.3，鸿蒙Next需要在卓易通环境下才能使用，请自行测试。
- ❌不支持Windows、iOS，感谢大家的理解。

### 如何下载/升级
- 在此仓库的Releases处选择需要的.apk安装文件下载即可。
- 腾讯频道：搜索【HyperFVM交流社区】，在【更新公告】版块可以找到最新的网盘链接，选择最新的.apk安装文件下载即可。
- App内：在【关于软件】中，点击【检查更新】按钮，可将App升级到最新版本。

### 版本选择
- Beta版：想要第一时间体验到新功能的同学可以选择Beta版，但需要注意可能会遇到未知Bug。
- Release版：推荐所有同学下载/更新。

### Bug反馈/功能建议
### 如果您有任何好的想法，欢迎通过以下方式反馈：
- 在此仓库发送相关Issue即可，有空就会回复，禁止发送任何与此App无关的内容。
- 腾讯频道：搜索【HyperFVM交流社区】，在【问题反馈】或者【功能建议】版块下发表帖子。
- 注意：看到基本都会回复，但不一定所有需求都会被满足；另外也请不要催促开发进度，作者也是利用自己的空闲时间为爱发电🌹
