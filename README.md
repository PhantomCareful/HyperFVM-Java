# 功能列表
### 已实现的功能
- **防御卡数据库**：除融合卡片外，其他所有卡片均已入库。输入卡片名称，即可查询其基础信息、星级数据、技能数据以及其他大部分有关此卡片的数据信息
- **可能是最全面的数据图合集**：内置So陌路の大佬制作的所有数据图，以及作者自己制作的一些数据图，方便随身查看
- **温馨礼包自动领取**：可以不限数量绑定温馨礼包链接，并可设置让App每天自动领取
- **信息仪表盘**：自动获取并在App内展示重要信息，包括：
  - ***双倍双爆信息***
  - ***月签道具数量提醒***
  - ***温馨礼包领取状态***
  - ***施肥活动信息***
  - ***美食悬赏活动信息***
  - ***重要时间点提醒***
    - *周三提醒关注更新公告*
    - *周四提醒更新维护*
    - *每月最后一天提醒清空积分和金券*
  - ***小彩蛋***：每年8月为美食妙妙屋公会庆生🎂
- **各种实用网站入口**
  - ***提拉米鼠官网***
  - ***卡片鼠军对策表***
  - ***陌路の综合数据表***
  - ***FAA米苏物流***
  - ***强卡最优路径计算器***
  - ***宝石最优路径计算器***
  - ***妙屋专属：满勤公会点公示***
  - ***妙屋专属：公会任务值班同学时间表***
- **各种实用小功能**
  - ***今日运势**：一个快速滚动播放包含1-16星海星图片的GIF动图*
  - ***温馨礼包绑定***
  - ***威望储备计算器**：方便让威望币爆仓的同学们知道自己一共有多少威望*
  - ***查黑系统**：查询某个QQ号是否有行骗记录*

### 计划实现的功能
只展示大功能，排名不分先后
- 防御卡数据库：查询星座卡、生肖卡、金卡时支持展示相应的分解兑换信息
- 防御卡数据库：查询卡片时支持展示相应的配方、材料、技能书的掉落信息（如果有的话）
- 支持一堆在线功能，比如：App支持应用内检查更新、所有数据图不再内置App，均从仓库获取
- App分为“社区版”和“妙妙屋专版”：“社区版”功能和文案更通用，“妙妙屋专版”会比“社区版”多一些专属功能
- UI引擎适配Miuix

# 一些小细节
- 使用Material You组件构建UI界面，支持Monet主题
- 所有界面的顶栏、底栏和悬浮按钮实现模糊处理
- 完美适配底部小横条沉浸，不要像一些国民级App那样做得稀烂🙄
- 支持预见式返回动画，在原生Android14+、OneUI和ColorOS16等系统上能体验到非常丝滑的过渡动画

# 安装
### 支持的系统环境
- ✅Android：支持Android12-16
- ✅HarmonyOS：支持鸿蒙1-4，鸿蒙Next需要在卓易通环境下才能使用，请自行测试
- ❌iOS&Windows：暂不支持，敬请谅解

### 如何下载？
- 在此仓库的Releases处选择需要的.apk安装文件下载即可
- 注意：发布的.apk文件一般分为2种：
  - Beta版：代表新功能测试或者问题修复验证，可能存在未知Bug，请谨慎安装
  - Release版：代表新功能开发完毕或者问题修复完毕，推荐安装
